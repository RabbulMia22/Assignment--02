Question-1: What are the different ways to select an element in the DOM?

Answer:There are many way to select element in the dom 1. getElementById 2. getElementsByClassName 3. getElementsByTagName 4. querySelector 5. querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent?

Answer: innerHTML: It will return the content of the element with HTML tags. innerText: It will return the content of the element without HTML tags. textContent: It will return the content of the element with HTML tags.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique that involves adding event listeners to a parent element. The listener will fire whenever the event is triggered on the childElement.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a type of event propagation where the event first triggers on the innermost target element and then successively triggers on the parents of the target element

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: To create an element, you can use the document.createElement() method. To add an element, you can use the appendChild() method. To remove an element, you can use the removeChild() method.
